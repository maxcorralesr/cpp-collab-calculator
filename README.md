# cpp-collab-calculator

Mini Calculadora Colaborativa en C++
Proyecto colaborativo para aprender Git, GitHub, ramas, pull requests y resolución de conflictos, desarrollando una calculadora simple en C++.

Descripción del proyecto
Este proyecto implementa una mini calculadora en C++ con funciones básicas:

Suma

Resta

Multiplicación

División

El objetivo principal es practicar el uso de GitHub en equipo.
